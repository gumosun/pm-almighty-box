# Cartova Brand Guide — 設計規範參考（虛構示範）

> ⚠️ **Cartova 是完全虛構的公司**，此 CI 僅供示範。
> 色彩變數定義在同目錄的 `brand-tokens.css`；這份文件是給人（與 AI）讀的規範說明與使用原則，兩者需保持一致。
> 所有 HTML 報告、簡報、產出物請以此色系與語氣為準。

---

## 色彩系統

| 角色 | 名稱 | HEX |
|------|------|-----|
| **主色 / 強調色** | Cartova Teal | `#0D9488` |
| **深色背景 / Header** | Deep Slate | `#0F172A` |
| **主文字** | Ink | `#1E293B` |
| **次要文字 / 副標** | Slate | `#64748B` |
| **淺色背景** | Teal Mist | `#F0FDFA` |
| **分隔線 / 邊框** | Cloud | `#E2E8F0` |
| **白底** | White | `#FFFFFF` |

**不要使用：** 純黑 `#000000`（一律用 Deep Slate）；紅色系不做主視覺，保留給錯誤/警示語意

---

## 字型

- **中文**：Noto Sans TC（Regular 400 / Medium 500 / Bold 700）
- **英文**：Inter（同字重）
- Google Fonts 載入：`https://fonts.googleapis.com/css2?family=Inter:wght@400;500;700&family=Noto+Sans+TC:wght@400;500;700&display=swap`

---

## 背景與色調使用原則

- 內文與數據區塊一律白底或 Teal Mist 淺底，維持選物品牌的乾淨留白感
- Deep Slate 深色底只用於封面頁、章節分隔頁與 Header，營造層次而非壓迫
- Teal 主色用於強調數字、行動呼籲與圖表主 series，單頁面出現面積不超過 15%

---

## 設計模式（HTML 報告適用）

| 元素 | 規範 |
|------|------|
| 標題層級 | H1 用 Deep Slate 粗體；H2 帶 Teal 左邊框 4px |
| 數據強調 | 關鍵數字用 Teal + Bold，輔以 Slate 說明文字 |
| 表格 | 表頭 Deep Slate 底白字，斑馬紋用 Teal Mist |
| 卡片 | 白底 + Cloud 1px 邊框 + 8px 圓角，陰影輕（0 1px 3px rgba(15,23,42,.08)） |

---

## 語氣（文案）

- 對內文件：直接、數據先行、結論在前
- 對會員文案：像懂生活的朋友，不像賣場廣播——「你上次買的洗手乳，差不多該補了」而非「限時優惠最後一天！」
